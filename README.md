# Raise Hand Momentum
Teams の会議で **他の参加者が挙手したら自動的に自分も挙手する** Tampermonkey 用ユーザースクリプトです。<br>
同調圧力に乗り遅れません。

## 概要

会議中に「参加者一覧」パネルを開くと監視を開始し、設定した人数以上の他参加者が挙手していると、自分も自動で挙手します。<br>**参加者一覧パネルの表示/非表示が、そのままスクリプトの ON/OFF スイッチ**になります。

## 主な機能

- **自動挙手**: 他の参加者の挙手者数がしきい値以上になると、自分も挙手する
- **ON/OFF は参加者一覧パネルで切り替え**: パネルを開いている間だけ動作（閉じると停止）
- **しきい値を画面で調整**: 左下のパネルの `−` / `＋` ボタンで「他 N 人以上で挙手」を変更（`localStorage` に保存され、次回も復元）
- **状態表示**: 左下に「監視中 / 他の挙手人数 / 自分の挙手状態」を表示

## インストール

1. ブラウザに [Tampermonkey](https://www.tampermonkey.net/) を導入する
2. [RaiseHandMomentum.user.js](https://github.com/h4ribote/raise-hand-momentum/raw/refs/heads/main/RaiseHandMomentum.user.js) を Tampermonkey に追加する

## 使い方

1. Teams 会議に参加する
2. ツールバーの **「参加者」** を押して参加者一覧パネルを開く → 監視開始（左下にステータス表示）
3. 左下パネルの `−` / `＋` でしきい値を調整する
4. 他の参加者がしきい値以上挙手すると、自動で自分も挙手する
5. 参加者一覧パネルを閉じると監視を停止する

## 設定（スクリプト冒頭の `CONFIG`）

| 項目 | 既定値 | 説明 |
|---|---|---|
| `pollMs` | `1500` | 監視間隔（ミリ秒） |
| `requireOthers` | `1` | しきい値の初期値（画面の `−`/`＋` で変更可） |
| `cooldownMs` | `3000` | 操作後のクールダウン（連打・誤検知防止） |
| `autoLower` | `true` | 他の全員が手を下げたら、自動で挙げた手も下げる |
| `lowerOnDisable` | `false` | 参加者一覧を閉じた（OFF）時に、自動で挙げた手を下げる |
| `showStatus` | `true` | 左下のステータス/操作パネルを表示する |
